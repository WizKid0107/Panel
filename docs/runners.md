# Runners

The panel displays artifacts. Runners prepare artifacts.

Examples:

- `html`: serve local HTML directly
- `markdown`: render Markdown
- `code`: render a code viewer with line numbers
- `image`: show raster/vector images
- `react`: start a Vite preview for JSX/TSX projects
- `diff`: render side-by-side or unified diffs
- `logs`: render searchable command/test logs

Keeping runners separate avoids turning the panel into one giant app.
